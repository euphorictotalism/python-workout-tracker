## Features

- Add new workouts
- Save workouts to a CSV file
- View workout history by exercise
- Track workout progress
- Display:
  - Starting weight
  - Latest weight
  - Maximum weight
  - Weight improvement

## Technologies

- Python 3
- CSV file storage
- File handling
- Functions
- Loops
- Conditional statements

## Example

Main Menu

1. Add a new workout
2. View workout profiles
3. Show progress
4. Exit

Example workout entry:

Date: 2026-06-29
Exercise: Bench Press
Muscle Group: Chest
Sets: 3
Reps: 8
Weight: 135
